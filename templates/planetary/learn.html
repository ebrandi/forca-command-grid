{% translate "Why bother?" %}

{% translate "Income while you sleep" %}

{% blocktranslate %}PI is the closest thing EVE has to passive income. Once a colony is set up it keeps producing; you just reset extractors and haul. A handful of planets can quietly fund your PLEX, your ammo, or a corp's fuel supply. It scales from "one planet, ten minutes a week" to "a factory network feeding capital construction".{% endblocktranslate %}

{% translate "Two kinds of planet" %}

{% translate "Extraction vs factory" %}

  • {% blocktranslate %}Extraction planet — an Extractor Control Unit pulls a raw P0 resource; Basic Industry Facilities refine it into P1. You export P1 (or feed it onward). Its output is set by how you place extractor heads and how often you reset.{% endblocktranslate %}
  • {% blocktranslate %}Factory planet — no extraction. It imports cheaper materials and combines them into higher-tier goods (P2, P3, P4). Its output is set by how many factories you can fit and feed.{% endblocktranslate %}
