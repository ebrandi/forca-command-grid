How to get this value

Discord → Server Settings → Integrations → Webhooks → New Webhook. Pick the target channel, Copy Webhook URL, and paste it above.

Easiest: create an Incoming Webhook at api.slack.com/apps → your app → Incoming Webhooks, and paste the hooks.slack.com/services/… URL. For DMs to pilots, set a bot token (PINGBOARD_SLACK_BOT_TOKEN) in the environment instead and give a channel id.

Create a bot with @BotFather → /newbot and copy its token into Telegram bot token above. Add the bot to your group/channel, then read the chat id (e.g. via @RawDataBot or the Bot API getUpdates) and paste it. Everything is stored here — no server access needed.

Pick Meta Cloud API or Twilio and paste that provider's credentials from its console — access/auth token above, ids/numbers in the fields. All stored here, encrypted; no server access needed. Note: outside a 24‑hour user‑initiated window both backends require a pre‑approved message template.
