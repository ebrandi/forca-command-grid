Export from EVE

  1. Open the Fitting window in the EVE client.
  2. Right-click a saved fit (or the browser) → Export to file. Selecting several exports them all into one XML.
  3. Upload that .xml here.

What happens next

New doctrines

Fits not already in the library — imported under the IMPORTED category.

Identical

Same name, hull and fit as an existing doctrine — skipped automatically, no change made.

Conflicts

Same name & hull but a different fit — you choose to skip, rename, or replace. Nothing is overwritten silently.

Invalid

Unknown ship/item, bad slot or quantity — rejected with a reason, never imported.

The file is parsed with hardened, offline XML processing (no external references or entity expansion) and is never stored or served. The database changes only when you confirm on the final step.
