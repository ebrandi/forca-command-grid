1 · {% translate "Blueprints: BPO vs BPC" %}

{% blocktranslate %}A blueprint is the recipe for building something. There are two kinds:{% endblocktranslate %}

  • {% blocktranslate %}BPO — Blueprint Original. Reusable forever. You buy it once (from CCP-seeded market orders for T1) and can build from it endlessly, and research it to make it more efficient.{% endblocktranslate %}
  • {% blocktranslate %}BPC — Blueprint Copy. A copy with a limited number of runs. Made by copying a BPO, or produced by invention (that's how you get T2/T3 blueprints — there are no T2 BPOs).{% endblocktranslate %}

2 · {% translate "Manufacturing, ME and TE" %}

{% blocktranslate %}Manufacturing turns materials into the product using a blueprint, in an industry job that takes time and costs a job fee. Two blueprint attributes matter:{% endblocktranslate %}

  • {% blocktranslate %}ME — Material Efficiency (0–10). Each level cuts the materials a job consumes by 1%, up to 10% at ME 10. Higher ME = cheaper builds. Research it on a BPO before mass production.{% endblocktranslate %}
  • {% blocktranslate %}TE — Time Efficiency (0–20). Each level cuts the job time by 1%, up to 20%. It doesn't change materials or cost, only how long the job runs.{% endblocktranslate %}
{% url 'industry:calculator' as calc_url %}

{% blocktranslate %}The Manufacturing calculator applies ME for you and shows the material bill after efficiency.{% endblocktranslate %}

3 · {% translate "Copying and research" %}

{% translate "Three more blueprint jobs, all run on a BPO:" %}

  • {% blocktranslate %}Copying makes BPCs from a BPO (needed as the input to invention).{% endblocktranslate %}
  • {% blocktranslate %}ME research raises Material Efficiency; TE research raises Time Efficiency.{% endblocktranslate %}

4 · {% translate "T2 invention" %}

{% blocktranslate %}Invention is how you make Tech II. You take a T1 BPC, add datacores (and optionally a decryptor), and run an invention job that might succeed. On success you get a T2 BPC with a few runs.{% endblocktranslate %}

  • {% blocktranslate %}Datacores — consumable science items, spent whether the job succeeds or fails.{% endblocktranslate %}
  • {% blocktranslate %}Decryptor — an optional modifier that trades success chance for more runs / better ME / TE on the resulting BPC.{% endblocktranslate %}
  • {% blocktranslate %}Probability — the chance a single job succeeds. It rises with your science + encryption skills and the decryptor. If a job has a 34% chance, you'll average about 3 attempts per success — so the real cost per T2 BPC includes the datacores you burned on the failures.{% endblocktranslate %}
{% url 'industry:invention' as inv_url %}

{% blocktranslate %}The T2 invention planner does this maths — probability, expected attempts, and cost per successful BPC — and tells you whether it's cheaper to invent, buy the BPC, or just buy the finished T2 item.{% endblocktranslate %}

5 · {% translate "What a job costs" %}

{% blocktranslate %}Building isn't just materials. A manufacturing job also charges an installation fee based on:{% endblocktranslate %}

  • {% blocktranslate %}Estimated Item Value (EIV) — CCP's reference value of the materials.{% endblocktranslate %}
  • {% blocktranslate %}System cost index — how busy that solar system's industry is (busier = pricier).{% endblocktranslate %}
  • {% blocktranslate %}Facility tax — set by whoever owns the structure you build in.{% endblocktranslate %}

{% translate "Our calculator estimates the fee from CCP adjusted prices × the cost index + facility tax. Structure and rig bonuses aren't modelled, so treat it as a close guide." %}

6 · {% translate "Profit, honestly" %}

{% translate "Your profit is what you sell for, minus everything it took to make and move it:" %}

{% translate "net profit = sale price − sales tax − broker fee − materials − job fee − invention − hauling" %}

  • {% blocktranslate %}Sales tax and broker fee are taken when you sell on the market.{% endblocktranslate %}
  • {% blocktranslate %}Margin is profit as a % of the sale price; break-even is the lowest price you can sell at without losing ISK.{% endblocktranslate %}

{% blocktranslate %}Every number this tool shows is an estimate from current Jita prices and your assumptions — not guaranteed profit. Prices move. Always sanity-check before committing ISK.{% endblocktranslate %}
